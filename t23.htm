Бесплатная безлимитная простая в использовании капча (captcha) для Ваших веб-проектов

Открытое API капчи без ограничений (мощность ограничена возможностями хостинга и ПО). Выводит число от 1000 до 9999 в HTML-разметке (картинка image, можно растягивать или сжимать с помощью CSS). Ключ расположен в поле key. Система не имеет настроек и не собирает Ваши данные при пользовании сервисом. Данные ответа API отправляются как text\plain JSON. Для PHP 8.1 работает скрипт примера (требует наличия пакета php-curl в системе):
<?php
$curl = curl_init();
curl_setopt($curl, CURLOPT_URL, "https://форумлюбви.рф/u");
curl_setopt($curl, CURLOPT_RETURNTRANSFER, 1);
curl_setopt($curl, CURLOPT_SSL_VERIFYHOST, 2);
curl_setopt($curl, CURLOPT_SSL_VERIFYPEER, 1);
$jsonAsPlainText = curl_exec($curl);
curl_close($curl);
$jsonArray = json_decode($jsonAsPlainText, true, 1, 0);
echo $jsonAsPlainText;
?>
